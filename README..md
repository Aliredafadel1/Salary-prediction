# Salary Prediction Project

This project predicts data science salaries using a Decision Tree model.  
It includes data preprocessing, model training, evaluation, and saving.

## Features
- Data preprocessing (handling missing values, encoding categorical data)
- Machine Learning model (Decision Tree Regressor)
- Model evaluation (MAE, RMSE, R²)
- Model saved for later use

## Current Progress
- Local dataset loading
- Data cleaning & preprocessing
- Model training
- Model evaluation
- Model saving

## How to Run

```bash
pip install -r requirements.txt
python train_model.py
